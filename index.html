Live site | Github Machblitz. JavaScript | Three.js | Blender | Substance3D/SubstancePainter | Substance Designer | Photoshop | Webpack | CSS. Racing game leveraging Three.js library and my past 3D experience. This was a good learning opportunity to better understand obj / fbx file formats, the Blender exporter, and the Three.js importer. Machblitz ultimately served as my JavaScript break-in project and was originally intended to be a quaternion visualization tool. While not a learning tool for others, quaternions are still used for the camera and vehicle motion so please check out the github.
